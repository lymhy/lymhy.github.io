随身wifi刷机入门指南

发表于2024-08-26|更新于2025-03-09|刷机指南
|浏览量:

​
前言:本教程适用于高通骁龙芯片的随身WiFi（简称棒子）的教程，不适用其他芯片和大厂棒子型号。

刷机前检查棒子功能是否正常，部分文件可能会报毒，事先添加系统的白名单或关闭实时保护。

1.打驱动。安装adb和9008驱动，双击安装启动文件即可。
adb运行方法是点击文件夹路径栏空白处输入cmd回车。

运行后后续操作可能发生冲突，进入设备管理器更换或卸载adb驱动即可。

2.备份系统包。棒子插入电脑，开机后在adb窗口输入 
adb devices

  有显示代表连接正常，继续下一步操作。

进入9008，有两个方法。
  一是adb窗口输入
adb reboot edl 

  二是按住棒子强刷键进入9008。
      tips:如果adb损坏可以尝试第二种方法，如果没有备份包的话一般来说救不回来了。

miko篇
  安装miko 运行miko loader，安装后桌面的快捷方式没有用可以删掉。

  确保棒子在9008，按照图示步骤操作。

保存好bin文件。卡0多试几次重新9008，检查设备供电是否正常。

恢复教程

Qualcomm Premium Tool 篇
使用软件需要注册，注册码输入位置

菜单栏找到help→active

注册好软件按照图示操作

root篇
打开ardc，重新插拔一下棒子，等待棒子启动，拖入窗口内准备好的安装包。等待一会后鼠标右键更换桌面。把Qualcomm Premium Tool备份的boot拖入ardc，进入Magisk修补boot。

进入adb窗口输入以下命令 

adb pull /sdcard/Download/magiskboot.img C:/xxx
自己更换一下路径img后面的路径

刷入修补boot，adb窗口输入：

adb reboot bootloader
 进入fastboot

然后输入命令：

fastboot flash boot
后再按个空格键，接着把电脑上magiskboot.img拖到这里，会自动生成文件路径名 

重启棒子使用ardc进入桌面检查面具是否刷入，这时候手机助手就没用了可以卸载掉。

基带篇
打开ARDC的>>这里的cmd后输入：

adb shell su
，授予shell永久权限，可在面具里查看。

关闭ardc和adb，打开星海强开901D，步骤如图所示。

备份qcn，流程如图，小于500k一般是没备份全。

3.刷机- Debian
棒子可以刷多个系统，但本教程只讲Debian。

两个方法

一：常规流程
下载base和Debian文件并解压。

文件地址

进入base文件夹，在路径栏空白处输入cmd回车

插入棒子等待棒子完全启动。

输入

adb reboot bootloader
回车进入fastboot

    双击base文件夹里面的flash bat按照提示进行

    进入Debian文件夹双击同样双击flash bat等待程序跑完

    重新插拔棒子电脑显示handsome外设则大功告成

如果电脑没有正确显示设备

   方法一：驱动程序→更新驱动程序→浏览我的电脑→网络适配器→Microsoft→基于远程ndis设备→无视        风险确定更新，重新插拔一下。

    方法二： 进入电脑设备管理器，双击新出现的位置设备或者在  Android Device  下查看有没有 Android Composite ADB Interface 这个驱动，如果不是，右键 更新驱动程序 - 浏览我的计算机以查找驱动程序软件 - 让我从计算机上的可用驱动程序列表中选取(L) ，选择 Android Composite ADB Interface - 下一步

如果不需要插卡的话可以刷一个一键刷入版的删减版的Debian，减少内存占用。

二：连接设备
打开cmd输入ssh，没有提示下个ssh软件。

输入并回车

ssh user@192.168.68.1
连接不上改成跟棒子相同的IP地址或者重置删除ssh hosts文件

     然后输入y 回车 然后输入1 回车然后输入sudo -i 回车
     然后输入1 回车(输入1同样没有显示)
     输入以下命令进入图形界面联网，只能连2.4gWiFi

nmtui
更新一下依赖

apt-get update
至此Debian的安装已经完毕，笔者认为棒子Debian不适合安装图形界面，现在的棒子基本都是4GB了，如果不担心内存爆满的话可以尝试一下。
